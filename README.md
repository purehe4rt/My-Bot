# Что бот может?
- Заходить на сервер с приветствием в чате
- Следовать за игроком по команде `следуй за мной` и `стоп следовать`
- Копать блоки по команде `собери <ID блока> <количество>`
- Отдать нужный предмет по команде `дай <ID предмета> <количество>`

# Инвентарь
Теперь инвентарь бота можно просматривать через веб-интерфейс. Когда бот зайдет на сервере, перейдите по адресу `localhost:3000`

# Настройки
Бота теперь можно настраивать в `config.json`. В будущем будут добавляться новые типы настроек

# Требования
На вашем компьютере должен быть установлен `NodeJS`. Проверить его наличие: зайдите в командную строку и напишите `node`. Если он у вас установлен - переходим к установке скрипта

# Установка и запуск
- Скачайте скрипт, он находится во вкладке `Releases`
- Распакуйте архив в любую удобную для вас папку
- Зайдите в эту папку и нажмите `ПКМ`
- В контекстном меню выберите `Открыть в терминале`
- В терминале введите `node bot.js`

ВАЖНО! Перед запуском скрипта зайдите в одиночный мир и откройте его для сети с портом из конфигурации
